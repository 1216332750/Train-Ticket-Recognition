# 电子动车票PDF批量提取与重命名工具

基于 PyMuPDF + RapidOCR 开发，支持文本/图片型铁路电子发票批量识别、信息提取、自动排序、文件重命名与 Excel 汇总导出。

## 一、功能亮点
- 自动识别下载的PDF电子动车票
- 提取乘车人、出发/到达站、车次、日期、金额、发票号码等关键字段
- 按「乘车人 + 乘车日期」自动排序，同名票据归类整理
- 批量重命名文件，统一规范命名便于归档
- 一键导出结构化 Excel 明细表，可直接用于财务报销

## 二、文件结构
项目目录/
├─ pdfs/                # 待处理发票PDF放入此文件夹
├─ 重命名后发票/        # 处理后重命名文件自动生成在此
├─ 铁路发票精准提取结果.xlsx
└─ main.py

## 三、环境安装
pip install pymupdf pandas rapidocr_onnxruntime openpyxl

加速安装（清华镜像）：
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pymupdf pandas rapidocr_onnxruntime

## 四、使用步骤
1. 将所有铁路电子发票 PDF 放入 pdfs 文件夹
2. 运行 main.py
3. 查看结果：
   - 重命名文件：重命名后发票 文件夹
   - 数据汇总表：铁路发票精准提取结果.xlsx

## 五、配置说明
可在代码顶部修改路径与输出文件名：

INPUT_DIR = './pdfs'
RENAMED_DIR = './重命名后发票'
OUTPUT_FILE = '铁路发票精准提取结果.xlsx'

## 六、提取字段
序号、来源文件、发票类型、发票号码、开票日期、乘车人、身份证号、出发站、到达站、车次、乘车日期、发车时间、坐席等级、车厢及座位、金额(元)、电子客票号、购买方公司、税号

## 七、文件命名格式
序号-乘车人-金额-发票类型.pdf

示例：
1-张三-89.50-购票发票.pdf
2-李四-23.00-退票发票.pdf

## 八、注意事项
- 仅适用于铁路客运电子发票
- 图片PDF越清晰，OCR识别越准确
- 文件名请勿包含特殊字符：/ \ : * ? " < > |
- 处理异常文件会在控制台提示，可单独检查文件完整性
